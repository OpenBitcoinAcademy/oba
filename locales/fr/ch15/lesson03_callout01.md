Bitcoin Core supporte les descripteurs de sortie depuis la version 0.17 (2018) et les descripteurs Miniscript depuis la version 24.0 (2022). Quand vous appelez `listdescriptors` sur un portefeuille Bitcoin Core moderne, il renvoie l'ensemble complet des descripteurs que le portefeuille utilise. Vous pouvez exporter ces descripteurs et les importer dans un autre portefeuille pour recreer le meme ensemble d'adresses.
