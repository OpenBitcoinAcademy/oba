Un protocole controle par une seule base de code est un point de defaillance unique. Les bogues dans une implementation affectent chaque noeud qui l'execute. La diversite des implementations n'est pas un luxe. C'est une propriete de securite. Quand btcd, bcoin et Bitcoin Core s'accordent tous sur la validite d'un bloc, cet accord a plus de poids que l'opinion d'un seul programme. La diversite previent la monoculture, et la monoculture est l'ennemi de la resilience.
