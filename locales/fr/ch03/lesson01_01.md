## Qu'est-ce que Bitcoin Core ?

Bitcoin Core est l'implementation de reference du protocole Bitcoin. Satoshi Nakamoto a ecrit le logiciel original et l'a publie en janvier 2009. Depuis, des centaines de developpeurs ont contribue a son evolution.

Le code source est ouvert et disponible sous la licence MIT. N'importe qui peut le lire, l'auditer, le modifier et le distribuer. Le depot est heberge sur github.com/bitcoin/bitcoin.

Bitcoin Core n'est pas Bitcoin. Bitcoin est un protocole, un ensemble de regles qui definissent le fonctionnement du reseau. Bitcoin Core est un programme qui implemente ces regles. D'autres programmes peuvent implementer les memes regles de differentes manieres.
