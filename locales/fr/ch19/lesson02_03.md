## RGB en pratique

RGB supporte les jetons fongibles (RGB20, y compris les stablecoins), les jetons non fongibles (RGB21) et les contrats programmables. Tether a annonce des plans pour emettre USDT sur RGB.

Les noeuds Lightning compatibles RGB peuvent transferer des jetons a la vitesse Lightning. Chaque transaction d'engagement inclut une sortie ancrant l'etat RGB. Les jetons circulent a travers les canaux de paiement sans modifier le protocole Lightning. Le reseau Lnfi a lance cette integration sur le mainnet en 2025.

La LNP/BP Standards Association (une association suisse a but non lucratif) maintient les specifications et les implementations de reference RGB. Le protocole a atteint la version 0.12 en 2025, ajoutant les zk-STARKs pour une meilleure vie privee et auditabilite.
