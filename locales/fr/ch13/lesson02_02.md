## Diversifier le risque

Ne stockez pas tous vos bitcoins dans un seul portefeuille. Separez les fonds par fonction : un portefeuille chaud (logiciel, connecte a internet) pour les depenses quotidiennes avec de petits montants, un portefeuille froid (materiel, hors ligne) pour l'epargne.

Les portefeuilles multisignatures necessitent plusieurs cles pour autoriser une transaction (2-of-3, 3-of-5). Distribuez les cles entre les appareils et les emplacements. Si une cle est compromise ou perdue, les fonds restent en securite. Le multisig ajoute de la complexite mais elimine les points de defaillance uniques.

Pour les organisations, le multisig avec des politiques de gouvernance (qui detient quelles cles, quel processus d'approbation est requis) fournit une responsabilite et une securite au-dela de ce qu'une seule personne peut atteindre.
