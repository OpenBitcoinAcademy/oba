## La racine de confiance

Chaque systeme de securite a une racine de confiance : le seul composant qui doit etre fiable pour que tout le reste fonctionne. Dans une banque, la racine de confiance est la banque elle-meme. Dans Bitcoin, la racine de confiance est les regles de consensus appliquees par votre noeud complet.

Si vous executez votre propre noeud complet, vous verifiez chaque regle independamment. Votre racine de confiance est le logiciel sur votre propre materiel. Si vous dependez du noeud de quelqu'un d'autre, votre racine de confiance se deplace vers ce tiers.

Les portefeuilles materiels ajoutent une seconde racine de confiance pour la securite des cles : la puce resistante aux manipulations qui detient vos cles privees. Le noeud complet verifie la blockchain. Le portefeuille materiel protege les cles. Ensemble, ils forment un modele de securite ou aucun point de compromission unique ne perd tout.
