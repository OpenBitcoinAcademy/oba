Qu'est-ce qui distingue un soft fork d'un hard fork ?

- Un soft fork necessite la mise a jour de tous les noeuds, pas un hard fork
- Un soft fork resserre les regles pour que les anciens noeuds suivent toujours la chaine, tandis qu'un hard fork rompt la compatibilite avec les anciens noeuds
- Un soft fork est propose par les mineurs, tandis qu'un hard fork est propose par les developpeurs
