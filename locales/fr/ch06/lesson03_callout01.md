L'astuce du marqueur/drapeau segwit est retro-compatible. Les analyseurs legacy lisent le marqueur (0x00) comme "zero entrees", ce qui est invalide, et rejettent donc les donnees. Les analyseurs compatibles segwit reconnaissent le zero comme un signal pour le format etendu et continuent de lire le drapeau, les entrees, les sorties, le temoin et le locktime. Un seul octet distingue les deux formats sans casser les anciens logiciels.
