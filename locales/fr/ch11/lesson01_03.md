## Confirmations

Une fois qu'une transaction est incluse dans un bloc, elle a une confirmation. Chaque bloc suivant ajoute au-dessus augmente le nombre de confirmations.

Une confirmation signifie qu'un mineur a inclus votre transaction. Six confirmations (environ une heure) est le seuil traditionnel pour les paiements de grande valeur. A cette profondeur, annuler la transaction necessiterait de reecrire six blocs, ce qui demande plus de puissance de calcul qu'un attaquant n'est susceptible de posseder.

Pour les petits paiements, une ou deux confirmations suffisent. Certains commercants acceptent les transactions a zero confirmation pour les achats de faible valeur, acceptant le faible risque que la transaction puisse etre remplacee avant confirmation.
