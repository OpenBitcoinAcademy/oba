Frostsnap est un projet open source de signature materielle construit specifiquement pour les signatures a seuil FROST sur Bitcoin. Il utilise plusieurs appareils materiels de coordination, chacun detenant une part de cle, pour signer les transactions. Le projet demontre que FROST est pratique pour la garde reelle aujourd'hui, pas une possibilite theorique lointaine. Les appareils de signature communiquent via un bus partage pour completer le protocole en deux tours sans exposer aucune part secrete a un ordinateur generaliste.
