## Warum Mining Sicherheit bietet

Miner können Belohnungen nur verdienen, indem sie Blocks erzeugen, die den Konsensregeln folgen. Ein Block mit ungültigen Transactions wird von jeder Full Node im Netzwerk abgelehnt. Der Strom des Miners ist verschwendet und er verdient nichts.

Das erzeugt einen wirtschaftlichen Anreiz zu ehrlichem Verhalten. Ehrliches Mining ist profitabel. Unehrliches Mining verschwendet Ressourcen. Je mehr Rechenleistung auf ehrliches Mining entfällt, desto schwerer wird es für einen Angreifer, betrügerische Blocks zu erzeugen.

Um eine bestätigte Transaction rückgängig zu machen, müsste ein Angreifer den Proof of Work für den Block mit dieser Transaction und jeden darauf aufbauenden Block wiederholen. Jede zusätzliche Bestätigung macht eine Umkehrung exponentiell teurer.
