## Das Gesamtbild

Bitcoin funktioniert, weil Tausende unabhängige Nodes die gleichen Regeln ohne Koordination befolgen. Jede Node verifiziert jede Transaction und jeden Block eigenständig. Keine Node vertraut einer anderen.

Wallets verwalten Keys und erstellen Transactions. Das Peer-to-Peer-Netzwerk verbreitet Transactions und Blocks. Miner konkurrieren darum, Blocks durch Proof of Work hinzuzufügen. Full Nodes setzen die Regeln durch, indem sie alles Ungültige ablehnen.

Diese Komponenten bilden ein System, in dem Werte zwischen Menschen überall auf der Welt bewegt werden können, ohne Erlaubnis von oder Vertrauen in einen Dritten. Der Rest dieses Kurses geht auf jede Komponente im Detail ein.
