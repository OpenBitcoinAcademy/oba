## Wallets, Keys und Addresses

Ein Bitcoin Wallet ist eine Sammlung von Keys, kein Behälter für Coins. Die Wallet-Software verwaltet Private Keys und nutzt sie, um Transactions zu signieren. Die Blockchain verzeichnet, welche Outputs mit welchen Keys ausgegeben werden können.

Um Bitcoin zu empfangen, teilst du eine Address. Eine Address wird von deinem Public Key abgeleitet. Jeder kann Bitcoin an deine Address senden, aber nur du kannst sie ausgeben, weil das Ausgeben eine Signature vom zugehörigen Private Key erfordert.

Um Bitcoin zu senden, wählt dein Wallet unverbrauchte Outputs (UTXOs), die du kontrollierst, erstellt eine Transaction, signiert sie mit deinem Private Key und sendet sie ins Netzwerk.
