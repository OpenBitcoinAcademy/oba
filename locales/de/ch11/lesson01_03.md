## Bestätigungen

Sobald eine Transaction in einen Block aufgenommen wird, hat sie eine Bestätigung. Jeder weitere Block, der darüber hinzugefügt wird, erhöht die Anzahl der Bestätigungen.

Eine Bestätigung bedeutet, ein Miner hat deine Transaction aufgenommen. Sechs Bestätigungen (etwa eine Stunde) sind der traditionelle Schwellenwert für Zahlungen mit hohem Wert. In dieser Tiefe müsste ein Angreifer sechs Blocks umschreiben, was mehr Rechenleistung erfordert, als ein Angreifer wahrscheinlich aufbringen kann.

Für kleine Zahlungen reichen eine oder zwei Bestätigungen. Manche Händler akzeptieren Zero-Confirmation-Transactions für Käufe mit niedrigem Wert und nehmen das geringe Risiko in Kauf, dass die Transaction vor der Bestätigung ersetzt werden könnte.
