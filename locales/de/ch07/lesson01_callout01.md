Bitcoin Script ist absichtlich eingeschränkt. Es hat keine Schleifen, keine Rekursion und keinen Zugriff auf externe Daten. Jedes Script terminiert in begrenzter Zeit. Das ermöglicht es jedem Node, jede Transaction zu verifizieren, ohne das Risiko endloser Berechnungen. Trotz dieser Einschränkungen kann Script eine große Bandbreite von Bedingungen ausdrücken: Eigentumsbeweise, Timelocks, Multisignature-Anforderungen und Hash-Preimage-Challenges.
