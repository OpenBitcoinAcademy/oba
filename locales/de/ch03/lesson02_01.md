## Was eine Full Node tut

Eine Full Node lädt jeden Block in der Blockchain herunter und verifiziert ihn unabhängig, angefangen beim Genesis Block von 2009. Sie prüft jede Transaction gegen alle Konsensregeln. Sie pflegt ihre eigene Kopie des UTXO-Sets (aller unverbrauchten Transaction Outputs).

Wenn du eine Full Node betreibst, vertraust du niemandem. Deine Node verifiziert die gesamte Geschichte des Netzwerks eigenständig. Wenn ein Block oder eine Transaction irgendeine Regel verletzt, lehnt deine Node ihn ab, ungeachtet dessen, wie viele andere Nodes ihn akzeptieren.

Eine Full Node leitet auch gültige Transactions und Blocks an andere Nodes weiter und trägt so zur Gesundheit und Widerstandsfähigkeit des Netzwerks bei.
