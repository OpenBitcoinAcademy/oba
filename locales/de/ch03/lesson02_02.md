## Warum eine eigene Node betreiben?

Eine eigene Full Node zu betreiben hat vier Zwecke.

**Verifizierung ohne Vertrauen.** Deine Node validiert jede Transaction, die du empfängst. Du verlässt dich nicht auf einen Dritten, der dir sagt, ob eine Zahlung echt ist.

**Privatsphäre.** Wenn du die Node einer anderen Person nutzt, erfährt diese, welche Addresses zu deinem Wallet gehören. Deine eigene Node behält diese Information lokal.

**Das Netzwerk unterstützen.** Jede Full Node, die Transactions und Blocks weiterleitet, macht das Netzwerk dezentraler und schwerer zu stören.

**Zugang für Entwicklung.** Eine Full Node bietet eine programmierbare API zur Abfrage von Blockchain-Daten, zum Erstellen von Anwendungen und zum Testen von Transactions.
