## Ein flaches Netzwerk

Bitcoin ist ein Peer-to-Peer-Netzwerk. Jede Node verbindet sich direkt mit anderen Nodes. Es gibt keinen zentralen Server, keine Hierarchie und keinen besonderen Koordinator.

Alle Nodes sind protokolltechnisch gleichberechtigt. Jede Node kann jederzeit beitreten oder das Netzwerk verlassen. Keine einzelne Node kontrolliert das Netzwerk. Wenn eine Node offline geht, arbeiten die übrigen ohne sie weiter.

Diese flache Topologie ist eine bewusste Designentscheidung. Ein zentraler Server wäre ein einzelner Ausfallpunkt und ein Ziel für Zensur. Ein Peer-to-Peer-Netzwerk hat keine zentrale Schwachstelle.
