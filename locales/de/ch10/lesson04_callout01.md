Bloom Filter (BIP 37) gaben Wallet-Addresses an jede Node preis, mit der sich der Client verband. Wenn diese Node von einem Überwachungsunternehmen betrieben wurde, war die finanzielle Aktivität des Clients offengelegt. Compact Block Filters (BIP 157/158) beheben das, indem sie umkehren, wer den Filter hält. Auf Software umzusteigen, die BIP 157/158 unterstützt, ist eine spürbare Datenschutzverbesserung für alle, die ein Lightweight Wallet nutzen.
