## Eine sehr große Zufallszahl

Ein Bitcoin Private Key ist eine Zufallszahl, 256 Bit lang. Das ist das gesamte Geheimnis. Wer diese Zahl kennt, kontrolliert die damit verknüpften Bitcoin.

Es gibt keinen Benutzernamen, kein Passwort-Zurücksetzen, keinen Kundendienst. Der Private Key ist der einzige Eigentumsnachweis.

Einen Private Key zu erzeugen heißt, eine Zufallszahl zwischen 1 und einem Wert nahe $2^{256}$ zu wählen. Der Zahlenraum ist so riesig, dass zwei Menschen, die unabhängig voneinander Keys erzeugen, nie den gleichen wählen werden.
