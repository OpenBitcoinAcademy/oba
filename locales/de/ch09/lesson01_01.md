## Wer zahlt die Transaction Fee?

Der Absender zahlt die Transaction Fee. Die Fee ist keine separate Zahlung. Sie ist die Differenz zwischen dem Gesamtwert der Inputs und dem Gesamtwert der Outputs.

Wenn Alice eine Transaction mit Inputs von insgesamt 100.000 Satoshis und Outputs von insgesamt 99.500 Satoshis erstellt, sind die verbleibenden 500 Satoshis die Fee. Der Miner, der die Transaction in einen Block aufnimmt, kassiert sie.

Es gibt keine Regel, die eine Mindest-Fee vorschreibt. Miner bevorzugen aber Transactions mit höheren Fee-Raten pro Gewichtseinheit. Eine Transaction mit zu niedriger Fee kann Stunden, Tage im Mempool warten oder nie bestätigt werden.
