## Das Halving und die 21-Millionen-Grenze

Die Block-Subsidy begann bei 50 BTC pro Block im Januar 2009. Alle 210.000 Blocks (ungefähr vier Jahre) halbiert sie sich. Das erste Halving im November 2012 reduzierte die Subsidy auf 25 BTC. Das zweite im Juli 2016 senkte sie auf 12,5 BTC. Das dritte im Mai 2020 brachte sie auf 6,25 BTC. Das vierte im April 2024 reduzierte sie auf 3,125 BTC.

Diese geometrische Reihe konvergiert auf maximal 21 Millionen Bitcoin. Keine Protokollregel besagt direkt "21 Millionen". Die Obergrenze ergibt sich aus dem Halving-Zeitplan: $50 \times 210{,}000 \times \sum_{i=0}^{\infty} (1/2)^i = 50 \times 210{,}000 \times 2 = 21{,}000{,}000$. Etwa im Jahr 2140 wird die Subsidy auf null Satoshis abrunden.

Das Halving erzeugt eine vorhersagbare, sinkende Ausgabekurve. Jeder Teilnehmer kann das aktuelle Angebot verifizieren, indem er alle Coinbase-Outputs summiert. Keine Zentralbank, keine diskretionäre Geldpolitik.
