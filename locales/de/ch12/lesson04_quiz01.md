Was unterscheidet einen Soft Fork von einem Hard Fork?

- Ein Soft Fork erfordert, dass alle Nodes aktualisieren, während ein Hard Fork das nicht erfordert
- Ein Soft Fork verschärft Regeln, sodass alte Nodes der Chain weiter folgen, während ein Hard Fork die Kompatibilität mit alten Nodes bricht
- Ein Soft Fork wird von Minern vorgeschlagen, während ein Hard Fork von Entwicklern vorgeschlagen wird
