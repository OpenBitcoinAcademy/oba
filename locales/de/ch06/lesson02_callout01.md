Outputs unter einem bestimmten Wert werden als Dust bezeichnet. Sie kosten mehr Gebühren zum Ausgeben, als sie wert sind. Wenn du einen Change-Output von 100 Satoshis erzeugst, aber das Ausgeben eine Gebühr von 300 Satoshis erfordert, sind diese Coins praktisch verloren. Wallets vermeiden das Erzeugen von Dust-Outputs, indem sie winzige Restbeträge stattdessen zur Gebühr addieren.
