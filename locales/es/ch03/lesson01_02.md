## De cliente a plataforma

El software original de Bitcoin incluia una billetera, un minero y un nodo completo en un solo programa. Con el tiempo, estas funciones se separaron. La mineria paso a hardware y software especializado. Las funciones de billetera pasaron a aplicaciones dedicadas.

Bitcoin Core evoluciono hasta convertirse en una plataforma para operar nodos completos. Descarga y verifica toda la cadena de bloques, mantiene la base de datos UTXO, retransmite transacciones y bloques, y expone una API para consultar datos de la cadena de bloques.

La mayoria de los operadores de nodos completos en la red Bitcoin ejecutan Bitcoin Core. Establece el comportamiento de referencia contra el cual otras implementaciones se comparan.
