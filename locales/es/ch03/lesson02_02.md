## Por que ejecutar tu propio nodo?

Ejecutar un nodo completo cumple cuatro propositos.

**Verificacion sin confianza.** Tu nodo valida cada transaccion que recibes. No dependes de un tercero para saber si un pago es real.

**Privacidad.** Cuando usas el nodo de otra persona, esa persona conoce que direcciones pertenecen a tu billetera. Tu propio nodo mantiene esa informacion local.

**Apoyo a la red.** Cada nodo completo que retransmite transacciones y bloques hace la red mas descentralizada y mas dificil de interrumpir.

**Acceso para desarrollo.** Un nodo completo proporciona una API programable para consultar datos de la cadena de bloques, construir aplicaciones y probar transacciones.
