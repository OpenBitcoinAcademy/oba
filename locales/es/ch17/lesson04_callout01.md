Frostsnap es un proyecto de hardware de firma de codigo abierto construido especificamente para firmas umbral FROST en Bitcoin. Usa multiples dispositivos de hardware coordinados, cada uno con una porcion de clave, para firmar transacciones. El proyecto demuestra que FROST es practico para custodia real hoy, no una posibilidad teorica distante. Los dispositivos de firma se comunican a traves de un bus compartido para completar el protocolo de dos rondas sin exponer ninguna porcion secreta a un ordenador de proposito general.
