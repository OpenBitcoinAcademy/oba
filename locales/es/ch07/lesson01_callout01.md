Bitcoin Script es intencionalmente limitado. No tiene bucles, ni recursión, ni acceso a datos externos. Cada script termina en un tiempo acotado. Esto permite que cada nodo verifique cada transacción sin riesgo de computación infinita. A pesar de estas restricciones, Script puede expresar una amplia gama de condiciones: pruebas de propiedad, bloqueos temporales, requisitos de firma múltiple y desafíos de preimagen de hash.
