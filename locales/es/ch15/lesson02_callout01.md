El compilador de Miniscript optimiza para el tamano del testigo, no el tamano del script. Cuando multiples expresiones Miniscript codifican la misma politica, el compilador selecciona la que produce el testigo esperado mas pequeno al momento de gastar. El tamano del script se paga una vez al crear la salida, pero el tamano del testigo se paga cada vez que se gasta la salida. Optimizar para el tamano del testigo reduce el costo continuo de usar la politica.
