## La base de datos de la billetera

Una aplicación de billetera almacena sus datos en una base de datos. Esta base de datos contiene claves privadas, claves públicas, y metadatos como etiquetas de transacciones, notas de direcciones, y configuraciones de comisiones.

Algunas billeteras almacenan solo claves públicas y dependen de dispositivos externos (billeteras de hardware) para guardar las claves privadas y producir firmas. Otras almacenan todas las claves localmente. El modelo de seguridad difiere, pero el principio es el mismo: la base de datos de la billetera es el vínculo entre el usuario y la blockchain.

Las aplicaciones de billetera proveen una interfaz de usuario sobre la base de datos: muestran saldos, construyen transacciones, gestionan direcciones, y se conectan a la red Bitcoin.
