Tagged hashes sao uma medida de defesa em profundidade. Sem separacao de dominio, um atacante poderia encontrar entradas onde um hash TapTweak colide com um hash TapLeaf, potencialmente enganando o verificador a aceitar scripts invalidos. Tagged hashes tornam isso criptograficamente inviavel. Cada operacao de hash no BIP 340/341 usa uma tag unica: "TapTweak", "TapLeaf", "TapBranch", "TapSighash", "BIP0340/challenge".
