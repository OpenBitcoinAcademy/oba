## De Cliente a Plataforma

O software original do Bitcoin incluia uma carteira, um minerador e um no completo em um unico programa. Com o tempo, essas funcoes se separaram. A mineracao migrou para hardware e software especializados. As funcoes de carteira migraram para aplicativos dedicados.

O Bitcoin Core evoluiu para uma plataforma de operacao de nos completos. Ele baixa e verifica todo o blockchain, mantem o banco de dados de UTXOs, retransmite transacoes e blocos, e expoe uma API para consultar dados do blockchain.

A maioria dos operadores de nos completos na rede Bitcoin usa o Bitcoin Core. Ele define o comportamento de referencia contra o qual outras implementacoes se medem.
