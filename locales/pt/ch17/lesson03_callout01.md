O backup no ChillDKG requer duas pecas: a chave secreta de host do participante e os dados de recuperacao comuns. A chave secreta de host e um valor de 32 bytes que cada participante deve armazenar com seguranca. Os dados de recuperacao sao os mesmos para todos os participantes e podem ser armazenados por qualquer um deles ou por um terceiro nao confiavel. Perder ambas as pecas significa perda permanente do share daquele participante. Perder apenas os dados de recuperacao e recuperavel desde que t participantes ainda tenham suas chaves de host e qualquer copia dos dados de recuperacao exista.
