## A Flat Network

Bitcoin is a peer-to-peer network. Every node connects directly to other nodes. There is no central server, no hierarchy, and no special coordinator.

All nodes are equal in protocol terms. Any node can join or leave at any time. No single node controls the network. If one node goes offline, the rest continue operating without it.

This flat topology is a deliberate design choice. A centralized server would be a single point of failure and a target for censorship. A peer-to-peer network has no throat to choke.
