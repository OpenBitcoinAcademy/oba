## Address Formats

Bitcoin address formats have evolved over the years. Each generation is an improvement.

**Legacy (1...)** addresses start with the number 1. They use Base58Check encoding. This was the original format from 2009.

**Script (3...)** addresses start with the number 3. They enable more complex spending conditions, including multi-signature wallets.

**SegWit (bc1q...)** addresses use Bech32 encoding. They are lowercase, have better error detection, and result in smaller transactions with lower fees.

**Taproot (bc1p...)** addresses use Bech32m encoding. They offer improved privacy and enable more advanced smart contracts.

All four formats are valid and in use today. Newer formats are preferred for lower fees and better features.
