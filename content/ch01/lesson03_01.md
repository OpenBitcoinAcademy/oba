## A Very Large Random Number

A Bitcoin private key is a random number, 256 bits long. That is the entire secret. Whoever knows this number controls the bitcoin associated with it.

There is no username, no password reset, no customer support. The private key is the only proof of ownership.

Generating a private key means picking a random number between 1 and a value close to $2^{256}$. The number space is so vast that two people independently generating keys will never pick the same one.
