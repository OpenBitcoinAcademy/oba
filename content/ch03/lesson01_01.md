## What is Bitcoin Core?

Bitcoin Core is the reference implementation of the Bitcoin protocol. Satoshi Nakamoto wrote the original software and released it in January 2009. Since then, hundreds of developers have contributed to its evolution.

The source code is open and available under the MIT license. Anyone can read it, audit it, modify it, and distribute it. The repository lives at github.com/bitcoin/bitcoin.

Bitcoin Core is not Bitcoin. Bitcoin is a protocol, a set of rules that define how the network operates. Bitcoin Core is one program that implements those rules. Other programs can implement the same rules in different ways.
